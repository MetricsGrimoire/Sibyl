I done this with Python locally, here is a snip of code using v2.0 auth with http://example.com:5000/v2.0

from keystoneclient.v2_0 import client
keystone = client.Client(username="admin", password="Password_here", tenant_name="Admin", auth_url="http://api-example.client.acme.net:5000/v2.0")

this will get you in, then you need to do what ever it is you wanting to do.
