Hi,

I wanted to host a web server inside of my openstack server (running behind my home router) so I followed this guide: openstack.redhat.com/Neutron_with_existing_external_network

After that, I set the public network as "shared" (so my other projects can access it) and I set the private network as "shared". Then, I enabled DHCP on the public subnet so that my instances would get an IP if I hooked them up to the public network. I launched an instance of cirros and gave it ONLY the public network, and it got automatically assigned an ip of 192.168.1.50 (which is what I wanted).

Here is where the problem comes...when I try to ping 8.8.8.8 from inside cirros I am not able to do so. Also, when I look at my router's (192.168.1.1) web management, I am not able to see this instance as connected in the list. My question is, if it was given an IP of 192.168.1.50 from Nova's dhcp server, then shouldn't my external router also detect this?
